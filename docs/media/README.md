# Демо-материалы

Демо записано на iPhone в реальном чате ВКонтакте, без монтажа.

## Файлы

| Файл | Размер | Назначение |
|------|--------|------------|
| `IMG_9299_part1.webp` | ~2 МБ | Анимированное превью: отправка шифра (0:03–0:23) |
| `IMG_9299_part1.mp4`  | ~1.4 МБ | Тот же фрагмент в MP4 для скачивания |
| `IMG_9299_part2.webp` | ~0.6 МБ | Анимированное превью: расшифровка (0:31–0:41) |
| `IMG_9299_part2.mp4`  | ~1.1 МБ | Тот же фрагмент в MP4 для скачивания |

WebP используется для inline-превью в README (GitHub рендерит анимированный WebP, в отличие от `<video>`). MP4 — для тех, кто хочет оригинальное качество.

## Почему два фрагмента

Полная запись — 60 секунд. Самые показательные моменты:

- **0:03–0:23** — пользователь печатает сообщение, нажимает 🔒, шифр уходит в чат.
- **0:31–0:41** — получатель видит входящий шифр, расширение его расшифровывает, появляется переключатель `[шифр]/[текст]`.

Остальные 40 секунд — паузы, скроллы, повторные отправки. Их пропустили ради краткости.
